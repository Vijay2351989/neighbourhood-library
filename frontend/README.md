# frontend

Next.js 16 (App Router, TypeScript, Tailwind v4) staff UI for the Neighborhood Library — books CRUD, members CRUD, and a borrow/return/loans flow over gRPC-Web. The full setup, run, and test instructions live in the root [`README.md`](../README.md). For module-level design and rationale see [`docs/design/04-frontend.md`](../docs/design/04-frontend.md).
