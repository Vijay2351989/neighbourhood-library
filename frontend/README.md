# Frontend — Neighborhood Library

Next.js 16 (App Router, TypeScript, Tailwind v4) staff UI. Phase 1 ships only the scaffold and a placeholder home page; the real pages, gRPC-Web client, and TanStack Query wiring land in [Phase 6](../docs/phases/phase-6-frontend-mvp.md).

For project-wide setup see the [root README](../README.md). The target frontend layout — directory structure, page responsibilities, error UX, and currency formatting — is documented in [`docs/design/04-frontend.md`](../docs/design/04-frontend.md).
